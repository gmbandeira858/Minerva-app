# Gerador de Dados Sintéticos em Python

Script desenvolvido em Python para gerar dados fictícios de cadastro pessoal, incluindo informações como nome completo, data de nascimento, endereço, telefone, estado civil e dados familiares.

O objetivo do projeto é criar datasets simulados para uso em testes, desenvolvimento de dashboards e práticas de análise de dados.

## Tecnologias utilizadas

- Python
- Pandas
- Geração de dados sintéticos
- Exportação para Excel

## Funcionalidades

- Geração aleatória de nomes e sobrenomes
- Criação de datas de nascimento
- Simulação de estado civil e estrutura familiar
- Geração de dados de endereço e telefone
- Exportação automática para planilha Excel

## Como executar

1. Instale as dependências:

pip install pandas

2. Execute o script:

python gerar_dados.py

3. O arquivo será gerado automaticamente:

cadastro_homens.xlsx
